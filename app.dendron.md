# Dendron 

## Line wrap for code blocks
`F1 => Markdown Preview Enhanced : Customize CSS`
Add in the `style.less`:
```less
.markdown-preview.markdown-preview {
       pre, code {
            white-space: pre-wrap;
       }
}
```
