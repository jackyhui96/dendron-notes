
# AWK

## Sum up lines
```sh
awk '{ sum += $NF } END { if (NR > 0) print sum }'
```

## Sort by line length
```sh
awk '{ print length, $0 }' | sort -n -s | cut -d" " -f2-
```

## print line length
```sh
awk '{ print length }'
```

## Average of lines
```sh
awk '{ sum += $1; n++ } END { if (n > 0) print sum / n; }'
```
